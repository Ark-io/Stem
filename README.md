# Karma
> A IoT framework based P2P network.

![Logo](img/karma.png)

Karma是一个基于P2P网络，安全，多层次的物联网(IoT)框架。Karma的目标是提供一个不同于传统中心化设计的物联网方案，以试图解决传统中心化物联网方案中单点失效，数据安全，网络稳定性等方面的问题。
